GoFish

This is a basic game of go fish using a javascript AI as the opponent. The goal is to match your selected card with the oponents cards, when cards are matched they are added to your deck of collected cards. There is also an option to change the style of the deck, more decks could be added to the game easily by adding the image link and name to an array of objects.

Bash/PSQL finance script

A script written entirely from the command line using two files. Transactions.sh is a bash script that creates a command line interface with which you can interact with postgres database without using SQL and only use numerical inputs. Formatter.sh is a bash script that will format a csv file into clean data and insert that data into a postgres database. Formatter.sh has a sed command that will format the csv file, this sed command is tailored to format the csv file I was working with and may need to be edited to correctly format different data.
